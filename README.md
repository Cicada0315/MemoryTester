# MemoryTester
Fun game to challenge your brain!

# Install
### Clone the repository
```
git clone git@github.com:Cicada0315/MemoryTester.git
cd MemoryTester
```
### Install gem
```
bundle install
rails db:migrate
```
### Run
```
rails s
```
